Criando Textos

Textos são os famosos paragráfos que vemos no nosso dia, como no HTML que é representado por "p". Para criar isso, copie a pasta "script" e cole em seu projeto. Após isso indique o arquivo JS, dessa forma:

< script src="script/mth-text.js"> < /script>

Após indicar o JS, modifique o arquivo "mth-text.js" como quiser.
