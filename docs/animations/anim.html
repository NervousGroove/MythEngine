Utilizando Animações

Na MythEngine você pode criar animações facilmente. Existem duas opções! Você pode criar um sprite com um GIF ou carregar várias imagens para criar uma animação, que é o método padrão. Então nesse tutorial criaremos uma animação a partir do segundo método. Para isso copie a pasta "script" e cole em seu projeto. Após isso, copie o arquivo "animation.html" e cole no seu projeto. Agora, modifique a animação como quiser. Você também pode copiar o código do arquivo animation.html e colar no seu projeto.
