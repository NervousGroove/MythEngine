Eventos do Teclado

Você pode criar eventos no seu jogo a partir de cliques do teclado. Para isso, copie a pasta "script" e cole em seu projeto. Após isso indique o arquivo JS, dessa forma:

< script src="script/mth-keyboard.js"> < /script>

Após indicar o JS, modifique o arquivo "mth-keyboard.js" como quiser criando diferentes condições.
