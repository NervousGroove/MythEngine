Criando Alertas

Os alertas são a forma mais fácil de indicar erros ou conectar diretamente com o usuário através de um Prompt. Para criar um alerta, adidione a pasta "script" ao seu projeto e indique o arquivo JS dessa forma:


< script src="script/mth-alert.js"> < /script >


Após isso, modifique o arquivo "mth-alert.js" como quiser.
