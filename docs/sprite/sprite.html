Criando Sprites

Sprites são imagens que pode ser Backgrounds, Inimigos, alidados, personagens, armas, e qualquer coisa que seja visual. Para criar isso, copie a pasta "script" e cole em seu projeto. Após isso indique o arquivo JS, dessa forma:

< script src="script/mth-sprite.js"> < /script>

Após indicar o JS, modifique o arquivo "mth-sprite.js" e indique a URL da imagem. Essa URL pode ser da Web ou do seu computador.
